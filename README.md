This is a simple piece of code that shows:
1.) how to access file paths in NodeJS/Express
2.) how to upload to Cloudinary's database.

If you haven't heard it before, Cloudinary is a great service which lets you upload images and you can have urls to those uploaded images.Free.Handy for tiny prototype apps.

To run this code, you would need to run 

>npm install

Then run:

node app.js

Open your browser and head to:

localhost:5000
